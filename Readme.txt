In the "./just-in-time" and "./commit-generation" folders, run command "source run.sh" to start the fine-tuning process.

For JIT-DP, the dataset can be downloaded from https://github.com/jacknichao/JIT-Fine.
For CMG, the dataset can be downloaded from https://zenodo.org/records/7196966#.Y0juJHZBxmM.
For TYP, the dataset can be downloaded from https://github.com/giganticode/probes.
We provide the dataset for CCM and LTP in the folder "./datasets".
